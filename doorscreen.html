Doorscreen

Programmed by Hai Tran on 2 May 2020

About the Project

Project uses Ultrasonic sensor HC-SR04 to measure distance, DHT11 to measure temperature and 16x2 LCD screen to display

Application

The project can be placed in front of house or room. The display shows welcome message, shows the room or house temperature

and current time. It will only turned on if on object or person is detected, i.e distance below a certain amount

(I set it as 10 cm)
