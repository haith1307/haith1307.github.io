Which company to invest in?

Completed on 13 Jul 2021

About

The goal of the project is to pick out a few consistent, high-peroforming stocks among over 400 firms on the stock market. Process includes:

  • Data Collection: web crawling and scraping from Vietnamese stock website
  • Data Verification: randomly check whether collection has been correctly implemented
  • Feature Extraction: from data visualisation, engineer new features which has high correlation with labels
  • Analysis

Result

After implementation, a great number of misconceptions and false assumptions have been made. Therefore, the project is temporarily suspended. However, while doing some research, a keyword "Quantitative Trading" has come to mind, which may open doors to much larger project in the incoming future.
